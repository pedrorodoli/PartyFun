# PartyFun 🎉
Plataforma integral de juegos sociales inspirada en el estilo Jackbox, diseñada para la interacción en tiempo real en eventos y fiestas.

## 💻 Características Técnicas
- **Comunicación:** WebSockets mediante `Socket.io` para latencia mínima en juegos multijugador.
- **Backend:** Node.js con soporte para bases de datos relacionales (MySQL) y embebidas (SQLite3).
- **Seguridad:** Gestión de sesiones mediante `uuid` y configuración protegida con `dotenv`.

## 🌐 Acceso
Accede a la plataforma aquí: [https://partyfun.ddns.net/](https://partyfun.ddns.net/)

## 🛠️ Instalación
1. Configura el archivo `.env` basándote en `.env.example`.
2. `npm install`
3. `npm start`
